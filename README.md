# Data-Driven Insights into Vehicle Fuel Performance: A Telemetry Analysis

In this project, I engineered a data-driven analytical framework to quantify and predict the "Performance Gap" between manufacturer-provided laboratory fuel ratings and actual real-world vehicle fuel consumption. Utilizing a telemetry dataset of 600 trip-level records spanning 25 distinct vehicle models and multiple fuel configurations, I developed a comprehensive machine learning pipeline. 

I successfully built a predictive solution that empowers fleet managers to forecast fuel requirements with high precision, enabling a strategic shift toward optimized, data-driven vehicle procurement. I was able to run Multiple Linear Regression, Gradient Boosting, and Random Forest models, conducting feature importance analysis to reveal that manufacturer-rated base mileage predominantly drives real-world performance, significantly outweighing other dynamic operational factors. This transition from theoretical laboratory estimates to a data-driven reality allows organizations to efficiently allocate vehicle resources while minimizing the financial risks of performance volatility.

### Tech Stack
* **Languages:** Python
* **Libraries:** Pandas, Scikit-Learn, Seaborn, Matplotlib
* **Environment:** Jupyter Notebook
* **Models Deployed:** Multiple Linear Regression, Gradient Boosting Regressor, Random Forest Regressor
